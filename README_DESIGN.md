## ğŸ“ Need Help?

### Common Questions

**Q: How do I change the primary color?**
A: Look for `from-blue-600` classes and change to your desired color.

**Q: How do I adjust spacing between elements?**
A: Use gap and padding classes (gap-4, p-6, etc.)

**Q: How do I disable dark mode?**
A: Remove `dark:` prefixed classes from components.

**Q: How do I add more animations?**
A: Add new @keyframes in index.css and use them in components.

**Q: How do I make a component larger?**
A: Adjust padding (p-6), font size (text-lg), and spacing (gap-6).
